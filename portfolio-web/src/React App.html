SPEAK THROUGH PICTOS.

Picto sirve como una solucin nica para mejorar la comunicacin de las personas con discapacidades. La aplicacin utiliza pictogramas, conocidos como Pictos, que son smbolos grficos que representan palabras o conceptos. Al utilizar la fuente Open-Dyslexic, diseada especficamente para ayudar a las personas con dislexia, Picto asegura an ms que la comunicacin est optimizada para aquellos que enfrentan desafos de lectura. A travs de este proyecto, mi objetivo fue crear una interfaz intuitiva y visualmente atractiva que permita a los usuarios intercambiar pensamientos e ideas sin esfuerzo.

Este proyecto es un testimonio de mi compromiso de aprovechar la tecnologa para fomentar la inclusin y mejorar la vida de las personas con discapacidades. Al proporcionar una plataforma que permite una comunicacin eficiente a travs de la traduccin de Picto a palabras y de palabras a Picto, este proyecto capacita a los usuarios para expresarse de manera efectiva. Con su utilizacin de React e integracin de la fuente Open-Dyslexic, Picto encarna las mejores prcticas del desarrollo front-end y la accesibilidad.
